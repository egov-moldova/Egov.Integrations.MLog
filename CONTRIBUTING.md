# Contributing to Egov.Integrations.MLog

We welcome contributions! Please follow these guidelines to ensure a smooth process.

## How to Contribute

1.  **Report Bugs:** If you find a bug, please open an issue on GitHub with a clear description and steps to reproduce.
2.  **Suggest Features:** We're open to suggestions! Open an issue to discuss your idea.
3.  **Submit Pull Requests:**
    *   Fork the repository.
    *   Create a new branch for your changes.
    *   Ensure your code follows the existing style.
    *   Add tests for any new functionality.
    *   Submit a PR with a clear description of your changes.

## Development Setup

1.  Clone the repository.
2.  Ensure you have the latest .NET SDK installed (see `.csproj` for version).
3.  Run `dotnet restore` to install dependencies.
4.  Run `dotnet build` to compile the project.
5.  Run `dotnet test` to execute tests.

## Code Style

*   Use consistent indentation (tabs/spaces) as found in the project.
*   Follow standard C# naming conventions.
*   Provide XML documentation for all public APIs.
