# AGENTS.md

This project is built and maintained with the help of AI agents.

## Guidelines for AI Agents

When working on this repository, please:

1.  **Follow the established code style:** Maintain consistency with existing naming conventions, indentation, and structure.
2.  **Ensure documentation is updated:** Any changes to public APIs must include updated XML documentation.
3.  **Run tests before submitting:** Ensure all existing and new tests pass.
4.  **Keep it minimal:** Avoid adding unnecessary dependencies or complexity.
5.  **Respect the repository structure:** Place new code in `src/Egov.Integrations.MLog` and tests in `src/Egov.Integrations.MLog.Tests`.

## Agent Interactions

We encourage the use of AI tools for:
- Refactoring
- Test generation
- Documentation
- Bug fixing

Please mention in your pull requests if an AI agent was used to generate or significantly modify the code.
